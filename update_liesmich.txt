Update:	01-Artikelsystem V 3.1.0 nach V 3.2.0
=============================================

!!! ACHTUNG !!!
Voraussetzung für dieses Update ist die Version 1.3.0 oder größer des 01ACP!
Sie können diese Version hier herunterladen: http://www.01-scripts.de/01acp.php
!!! ACHTUNG !!!

- Legen Sie ein Backup ihrer Datenbank an!
- Legen Sie ein Backup ihrer Daten via FTP-Programm an!

- Löschen Sie das Verzeichnis 01module/01article/images/icons/*

- Markieren Sie alle im Verzeichnis 01article/ enthaltenen Dateien und Unterverzeichnisse und
  laden Sie sie per FTP-Programm in das entsprechende Modul-Verzeichnis (01scripts/01module/.....) auf Ihren Webspace hoch.
  Überschreiben Sie dabei alle bereits existierende Dateien.

- Loggen Sie sich in den Administrationsbereich ein und wählen Sie auf der Startseite im linken Menü
  den Punkt "Module verwalten".
  Starten Sie den Update-Vorgang für das 01-Artikelsystem durch einen Klick auf das Zahnrad-Symbol.

- Wenn Sie die Erweiterung "Artikelteaser auf externer Seite ausgeben" verwenden, müssen Sie auch diese
  Erweiterung nun aktualisieren.
